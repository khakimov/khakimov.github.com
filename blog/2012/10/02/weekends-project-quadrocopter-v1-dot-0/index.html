Weekends Project: Quadrocopter v1.0

Was an interesting quest to build this thing. At the end I’ve almost gave up about even test drive it and couple weeks it just waited till my friend asked to try outside, because I’ve tried to test it at home and it wasn’t good idea (no pilot expirience, didn’t know how to controls work and tried to start with perfect vertical lift-off and of course it didn’t work and I thought something wrong with my hands/software/engines/controllers). An old story, we are afraid to fail at something, so it was really easy to give up than just once test it outside.

Anyway, I did it and it flies perfectly. Time to improve pilot/control skills and later will upgrade it with camera and maybe with 3d glasses.
